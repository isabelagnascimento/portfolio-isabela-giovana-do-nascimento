# 🎬 Experiência 01: A Corrida do Prompt

## 📝 Descrição do Projeto
Este projeto documenta a experiência prática "A Corrida do Prompt", cujo objetivo principal é desenvolver habilidades de Engenharia de Instrução (Prompt Engineering) para obter resultados de alta precisão utilizando Modelos de Linguagem (LLMs). O grande desafio foi gerar um texto e uma imagem específicos com um limite estrito de apenas 5 iterações.

Desenvolvido como parte da disciplina, o projeto aplicou metodologias ativas de avaliação. A cada ciclo de interação com a IA, utilizamos a **Taxonomia de Bloom** para avaliar criticamente as respostas (identificando ausências de contexto e desvios de tom) e a **Taxonomia de Fink** para integrar o aprendizado ao próximo comando, garantindo que o modelo ajustasse sua percepção para atingir o objetivo final.

## 🚀 Tecnologias Utilizadas
* **Modelos de IA:** Inteligências Artificiais Generativas de Texto e Imagem
* **Técnicas Aplicadas:** Engenharia de Prompt, Refinamento Iterativo, Restrição de Contexto
* **Ferramentas de Versionamento:** Git e GitHub

## 📊 Resultados e Aprendizados
O projeto alcançou os objetivos propostos dentro do limite de tentativas, demonstrando a eficácia da construção lógica de comandos.
* **Controle de Persona (Texto):** O modelo assumiu a persona de um pirata de 1650 com sucesso. O maior aprendizado foi aplicar restrições rígidas para evitar formatos indesejados (como cartas) e garantir termos comerciais favoráveis na negociação fictícia com o rei (partilha de 60/40).
* **Evolução Estética (Imagem):** Aprendemos a importância do vocabulário técnico para direcionar a IA. Para chegar ao estilo barroco de um astronauta em Marte, foi necessário evoluir o prompt adicionando palavras-chave como "exagero", "movimento" e "tensão entre opostos", criando o contraste visual necessário.

## 🔧 Como Executar
1. Clone este repositório em sua máquina local.
2. Acesse a pasta `engenharia-prompt`.
3. Abra o arquivo `Engenharia de Prompt - Experiência 01.pdf` para visualizar o registro completo da evolução dos prompts, os comandos exatos utilizados pelo trio e os resultados gerados pela IA.
